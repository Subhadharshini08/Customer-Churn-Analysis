# Customer Churn Analysis

## Project Overview
End-to-end data analysis project on Telco Customer Churn dataset 
using Excel, SQL, Python and Power BI.

## Tools Used
- Excel — Data cleaning and pivot tables
- MySQL — Data analysis queries
- Python — Exploratory Data Analysis (EDA)
- Power BI — Interactive dashboard

## Dataset
- Source: Kaggle — Telco Customer Churn
- Total Customers: 7032
- Features: 21 columns

## Key Findings
1. Month-to-month customers churn at 42.71% vs 2.85% for two-year contracts
2. Fiber optic customers have highest churn rate at 41.9%
3. Electronic check payment customers churn at 45.3%
4. Senior citizens churn at 41.7% — almost double non-seniors
5. Churned customers pay $13 more per month than active customers

## Project Structure
- Data/ — cleaned dataset
- SQL/ — analysis queries
- Python/ — EDA notebook
- PowerBI/ — dashboard file
